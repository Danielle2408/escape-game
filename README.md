# # 🎓 Escape Game – Évasion de la classe

> Un jeu d'évasion (escape game) entièrement jouable dans le navigateur, sur le thème d'une salle de classe dont il faut s'échapper en trouvant des indices et en déchiffrant un code secret.

---

## 📁 Structure du projet

```
projet/
├── jeu.html       # Structure et interface du jeu
├── jeu.css        # Styles visuels et animations
└── jeu.js         # Logique du jeu (timer, indices, vérification)
```

---

## 🎮 Fonctionnalités

- ⏱️ **Minuteur** de 5 minutes pour s'échapper avant la fin du temps
- 🔍 **7 objets interactifs** à explorer dans la salle de classe
- 📜 **Panneau d'indices** qui se remplit au fur et à mesure des découvertes
- 🔐 **Code secret à 4 chiffres** à déchiffrer pour ouvrir la porte
- 🔄 **Bouton Recommencer** pour relancer une nouvelle partie

---

## 🗂️ Objets de la salle

| Objet | Icône | Description |
|---|---|---|
| Table | 📚 | Contient peut-être un indice caché |
| Sac | 🎒 | À fouiller soigneusement |
| Tableau | 📝 | Des indices peuvent y être écrits |
| Bibliothèque | 📖 | Cherchez entre les livres |
| Globe | 🌍 | Un objet mystérieux |
| Corbeille | 🗑️ | Les déchets cachent parfois des secrets |
| Porte | 🚪 | L'objectif final — ouvrez-la avec le bon code ! |

---

## 🚀 Lancement du jeu

1. Clonez ou téléchargez le projet
2. Ouvrez `jeu.html` dans votre navigateur
3. Aucune installation requise — le jeu fonctionne en local

```bash
git clone https://github.com/votre-repo/escape-game.git
cd escape-game
# Ouvrez jeu.html dans votre navigateur
```

## 🎨 Technologies utilisées

- **CSS3** — Thème sombre, animations, effets hover
- **JavaScript (Vanilla)** — Logique du jeu, timer, événements
