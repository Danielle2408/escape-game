# 🎓 Escape Game – Évasion de la classe

Un jeu d'évasion jouable dans le navigateur : explorez une salle de classe, trouvez des indices et déchiffrez le code secret pour vous échapper avant la fin du chrono !

---

## 📁 Fichiers

| Fichier | Rôle |
|---|---|
| `jeu.html` | Structure de l'interface |
| `jeu.css` | Styles et animations |
| `jeu.js` | Logique du jeu (partiel — voir collaboration) |

## 🚀 Lancement

Ouvrez simplement `jeu.html` dans votre navigateur. Aucune installation requise.

## 🌿 Branche `feature/Html-css` — Commits

| # | Message |
|---|---|
| 1 | `feat: ajout de la structure HTML du jeu d'évasion` |
| 2 | `feat: ajout du style CSS (thème, objets, panneaux)` |
| 3 | `chore: liaison CSS/JS vérifiée et annotation de version` |
| 4 | `feat: ajout de la logique vérification du code et reset du jeu` |

## 👥 Collaboration

Le fichier `jeu.js` est **partagé**. Les fonctions `generateRandomCode`, `bindObjectClicks`, `startTimer`, `playSound` et `objectsConfig` sont à compléter par les autres branches.
